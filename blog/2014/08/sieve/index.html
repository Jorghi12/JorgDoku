Prime Sieves and Python Data Structures

Date Fri, 29 Aug 2014 Tags Python

Recently, I’ve been working through Project Euler problems in order to improve my core programming skills. More likely it’s just an excuse to play around and code some easy programs, but lately the harder problems have been challenging in a way that requires an efficient algorithm to solve in a managable amount of time and space.

One of those problems that seem to recur requires an understanding efficiently calculating and testing for prime numbers. One of the more simple and yet efficient solutions are sieves algorithms, the most famous likely being The Sieve of Eratosthenes. Simply, rather than computing all the permutations of factorizing a value, you start a list of all values and sieve away all multiples and eventually end up with a lit of only non-factorable units

Original Sieve: Unoptimized

def is_prime(n: int) -> list:
    """
    Find all the primes by perfoming modulo
    between all possible values.
        2 < x < n
    """
    for i in range(2, n+1):
        if n % i == 0:
            return False
    return True

def sieve(n: int) -> list:
    """
    Sieve away and only primes are left.
    """
    primes = 2*[False] + (n-1)*[True]
    for i in range(2, n+1):
        if not is_prime(i):
            for j in range(i*i, n+1, i):
                primes[j] = False
    return [prime for prime, check in enumerate(primes) if check]

Sets: Unordered Collections

def set_sieve(n: int) -> set:
    """
    Sets are unordered, but hashable collections
    which offer optimizations for membership testing
    at O(1) time complexity.
    """
    primes = set(range(2, n+1))
    for i in range(2,int(n**0.5+1)):
        primes -= set(range(i*i, n+1, i))
    return primes

Sets Optimized Insertion

def set_insertion_sieve(n: int) -> set:
    """
    Further optimized set implementation by
    performing insertion over deletion utilizing
    constant time nature of hashable collections.
    """
    factors = set()
    for i in range(2,int(n**0.5+1)):
        if i not in factors:
            factors |= set(range(i*i, n+1, i))
    return set(range(2,n+1)) - factors

Numpy

import numpy as np

def np_sieve(n: int) -> np.ndarray:
    """
    Numpy n-dimmensional array are continguous strides
    of memory with fixed width statically typed elements.
    Ndarrys are a  random access data-structure unlike
    pythons built-in doubly linked-lists.
    """
    primes = np.ones(n+1, dtype=np.bool)
    for i in np.arange(2, n**0.5+1, dtype=np.uint32):
        if primes[i]:
            primes[i*i::i] = False
    return np.nonzero(primes)[0][2:]

A Timer

import time
from contextlib import contextmanager

@contextmanager
def timer(label):
    start = time.time()
    try:
        yield
    finally:
        end = time.time()
    print('{label}: {time:03.3f} sec'.format(
        label=label, time=end-start)
    )
